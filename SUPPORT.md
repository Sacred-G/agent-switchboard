# Support

This document explains where to get help with Agent Switchboard and what information to include when asking for support.

## Before Asking for Help

Please check:

1. The root [README](README.md) for setup and common commands.
2. The user manual in `docs/user-manual/`.
3. Existing issues and discussions in the project repository.

Repository: https://github.com/farion1231/agent-switchboard

## Getting Help

Use the project repository to:

- Open a bug report for reproducible defects.
- Open a feature request for proposed improvements.
- Start or join discussions when you have questions about usage or configuration.

## What to Include

For faster troubleshooting, include:

- Agent Switchboard version or commit SHA.
- Operating system, version, and CPU architecture.
- Whether you installed a release build or are running from source.
- The assistant/tool involved, such as Claude Code, Codex, or Gemini CLI.
- Steps to reproduce the issue.
- Expected behavior and actual behavior.
- Relevant logs or screenshots.

Remove secrets before sharing logs or screenshots. Do not post API keys, tokens, private endpoints, or personal configuration values publicly.

## Development Support

If you are contributing code, see [CONTRIBUTING.md](CONTRIBUTING.md) for setup, checks, and pull request expectations.

## Security Reports

If you believe you have found a security vulnerability, do not open a public issue. Follow [SECURITY.md](SECURITY.md).
