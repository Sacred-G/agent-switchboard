Claude Configuration + URL Parameter Override

Configuration file provides basic settings, URL parameters override API Key. URL parameters have the highest priority.

Required name, config
Override apiKey (Override values in config file)
📥 Import Mixed Configuration
// Decoded configuration content:
{
  "env": {
    "ANTHROPIC_BASE_URL": "https://api.anthropic.com/v1",
    "ANTHROPIC_MODEL": "claude-sonnet-4.5"
  }
}
// URL parameter override: apiKey=sk-ant-my-new-key
Priority Rules: URL parameter (apiKey) > Configuration file (endpoint, model)

Codex TOML Configuration Import

Codex uses TOML format configuration file, containing auth and config sections.

Required name, config
Optional configFormat=json (Codex configured asJSON packagedTOML)
📥 Import Codex Configuration
// Decoded configuration content:
{
  "auth": {
    "OPENAI_API_KEY": "sk-proj-test-key123"
  },
  "config": "[model_providers.openai]\nbase_url = \"https://api.openai.com/v1\"\n\n[general]\nmodel = \"gpt-5.1\""
}
// config field parsing (TOML):
[model_providers.openai]
base_url = "https://api.openai.com/v1"
[general]
model = "gpt-5.1"
